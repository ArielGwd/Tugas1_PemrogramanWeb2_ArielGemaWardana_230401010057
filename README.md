# Bookshelf - App Mini CRUD (PHP)
adalah sebuah aplikasi mini crud tentang catatan buku yang dibaca oleh pengguna.  

## Feature
- Create
- Read
- Update
- Delete
- Search
- Pagination 

## Database Structure
```sql
Database: tugas_web2_arielgema 
table: books 
```
